Estructura de la aplicación:

Front-End

  • AngularJs
  • jQuery
  • Bootstrap Twitter
  • Uso de “hubs” mediante SignalR 2.2.0

Back-End

  • ASP.Net Web API 2 en la parte superior de Owin (Open Web Interface for .NET) middleware.
  • ORM: Entity Framework 6.0.0.0
  • Autenticación: Dentro del Owin middleware --> Emisión de token de autenticación OAuth + Identity 2.0.0.0 (para registrar usuarios y validar credenciales antes de emitir el token)
  • Motor de base de datos: SQL Server
  • Uso de “hubs” mediante SignalR 2.2.0
